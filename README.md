# 介紹
將網址縮短的功能。


## 功能
- 將網址縮短
- 複製短網址

## 開始使用
1. 請先確認有安裝 Node.js與npm
2. 在本地開啟後，進入終端機輸入：
  npm install
3. 安裝完畢後輸入：
  npm run dev
4. 若成功連接會顯示：
  APP.js is running on http://localhost:3000
5. 結束運行輸入：
control + c
